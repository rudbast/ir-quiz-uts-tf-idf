# IR Quiz Assignment [Oct 2015]

## How to

Run program in src/

```shell
perl indexing.pl
perl vsm-similarity-product.pl
```

or just change file permission to executable `chmod +x *` and run

```shell
./indexing.pl
./vsm-similarity-product.pl
```
